gver960 Version Tool

The gver960 utility displays version information about installed tools, by invoking them with the v960 option. The version information includes a version number and a build time/date stamp. The invocation command is:

gver960 [ -option ]... [ basedir ]

Each option must be preceded by a - character. (On DOS, you can also use a / character.) The options are:

  • h
    Prints help.
  • v960
    Prints version information about gver960 and exits.
  • v
    Prints out each command before executing it.

basedir is the tools' base directory, containing bin and related subdirectories. The basedir can also be specified with the G960BASE or I960BASE environment variable. If basedir is omitted, gver960 searches for these environment variables, in this order.
