gsize960/siz960 Section Size Printer

The gsize960/siz960 section size printer ("sizer") prints the number of bytes required for each section of an object file, and their sum. It also displays the address for each of the sections.

The invocation command is:

gsize960 | siz960 [ -option ]... [ file ]

Each option must be preceded by a - character. (On DOS, you can also use a / character.) The options are listed below. If no options are specified, sizes are displayed in decimal and addresses are displayed in hex.

file identifies an object file. If the file is an object library, the sizer reports the size for each member of the library. If file is omitted, the sizer reports the size of a.out, b.out, or e.out.

  • c
    Displays total size of COMMON symbols.
  • d
    Displays size in decimal radix (this is the default).
  • h
    Sends help information to stdout and exits.
  • o
    Displays size in octal radix.
  • p
    Suppresses header display.
  • V
    Sends version information to stdout and then processes the rest of the specified operations.
  • v960
    Sends version information to stdout and exits.
  • x
    Displays size in hex radix.
